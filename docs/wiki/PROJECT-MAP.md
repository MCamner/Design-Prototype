# Design-Prototype Project Map

Design-Prototype is a browser-based enterprise validation lab.

It groups lightweight prototypes for:

- endpoint readiness
- macOS compliance
- fleet visibility
- certificate expiry risk
- GUI-to-CLI operational handoff
- client optimization and reporting

## Prototype groups

### 1. Endpoint readiness
- Client Readiness Dashboard
- MQ Client Optimizer

### 2. Fleet visibility
- Fleet Command Center
- MQ Fleet Report

### 3. macOS enterprise validation
- macOS Enterprise Dashboard
- Certificate Expiry Timeline

### 4. Operator workflow tools
- MQ Mirror
- MQ Site Fix Advisor
- MQ Asset Downloader
- Draw.io Generator

## Publish goal

Each prototype should have:

- live demo link
- screenshot
- short problem statement
- quick start
- helper requirements
- safe sharing notes
- roadmap
