How to use MQ Site Fix Advisor
1
Enter a URL or paste HTML

Type a public page URL in the URL field and press Enter or click Run audit. If the page blocks browser fetching (CORS), copy the page source and paste it into the HTML textarea instead.

2
Pick a fetch mode if needed

Auto fallback tries four proxy services in order — it works for most pages. If a specific proxy is more reliable for your site, select it directly from the dropdown.

3
Choose which checks to run

Toggle SEO, Images, Links, and CSS checks on or off before scanning. Uncheck categories that are not relevant to your current task to reduce noise.

4
Read the results

Click any row in the issue table to see the explanation, recommended fix, code snippet, and platform-specific note in the right panel. Use the tabs (SEO / Images / Links / CSS) to filter by category.

5
Generate fix

Click Generate fix to get a corrected version of the page's <head> with all automatically fixable issues already applied — ready to copy and paste. Switch to Full HTML to get the complete fixed page source.

6
Export the report

Click Export JSON to download the full audit report as a JSON file — useful for sharing with a developer or keeping a record of the page's state before changes.
