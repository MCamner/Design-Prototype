# Design-Prototype Project Map

Design-Prototype is a browser-based enterprise validation lab for endpoint readiness, macOS compliance, fleet visibility, and operator workflows.

## Main prototype groups

### Endpoint readiness
Tools and dashboards for checking whether a client is ready for enterprise workflows.

- Client Readiness Dashboard
- MQ Client Optimizer

### Fleet visibility
Views for understanding endpoint state across multiple clients.

- Fleet Command Center
- MQ Fleet Report

### macOS enterprise validation
Browser-based views and helpers for macOS compliance, certificates, updates, MDM, users, and security posture.

- macOS Enterprise Dashboard
- Certificate Expiry Timeline

### Operator workflow tools
Small tools that help operators translate, document, validate, or package technical work.

- MQ Mirror
- MQ Site Fix Advisor
- MQ Asset Downloader
- Draw.io Generator

## Publish goal

Each prototype should have:

- clear problem statement
- live demo or local run instructions
- screenshot
- quick start
- known limitations
- safe sharing notes
- roadmap
